## 포트폴리오 관점 강점
1. **금융 계산 로직이 분명하다**
   - NPV / IRR / Payback / 시나리오 / 민감도 분석을 직접 설명 가능
2. **단순 CRUD 이상이다**
   - 평가 결과 이력 저장과 비교 시각화가 들어가 있어 시스템성이 보임
3. **화면 + API + 테스트가 연결되어 있다**
   - UI, REST API, 테스트 전략이 각각 따로 놀지 않음
4. **문서 설득력이 있다**
   - README, 스크린샷, 아키텍처 문서까지 포함

> 이 프로젝트는 금융상품 비교와 프로젝트 투자 타당성 평가를 하나의 흐름으로 통합한 포트폴리오입니다.
